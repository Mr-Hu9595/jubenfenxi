
# 总制片人版｜评估指标全面优化

目标：在不改变 Excel 填报结构（内容/制作/宣发）的前提下，系统性评估每部剧本的整体质量与商业潜力，强化“人物饱满度”与可量化指标，指导评估者客观打分、提出修改建议并预测市场表现。全程使用中文。

评估指标体系（完整口径）

一、内容（结构与文本层）

- 文量/集数：统计字数与页数；结合爽点密度与转折频率，建议集数区间（字数≤8k→15–20集；8–20k→20–30集；＞20k→30–40集）。
- 倾向：男频/女频/双频（依据关键词与叙事侧重，如系统/逆袭/打脸→男频；甜宠/虐恋/霸总→女频；混合元素→双频）。
- 时代背景：现代/古代/民国/科幻/其他（依据文本线索与标题）。
- 主题：提炼题材标签（都市/家庭/职场/武侠/悬疑/复仇/甜宠/乡村/系统/重生等）。
- 人物饱满度：衡量人物多面性与反差弧光（“反转/黑化/洗白/救赎/成长/挣扎/人性”等关键词与连接词“但是/然而/一方面/另一方面/看似/实则/表面/内心”及反差词对的加权）。
- 剧情结构：起承转合完整度、转折密度、悬念闭合率、节奏均衡度。
- 角色塑造：目标-动机-障碍框架、弧光与反差、关系网复杂度、记忆点（昵称/口头禅/独特行为）。
- 冲突设置：类型覆盖（人-人/人-环境/人-自我）、升级曲线、风险与代价、解决策略多样性。
- 台词质量：人设匹配、信息密度、金句率与记忆点、生活化/专业度、价值观与合规。

二、制作（落地与成本层）

- 导演（风格建议）：商业爽剧/写实都市/古装群像/武侠动作/悬疑快剪。
- 场景数量：估算总场景数与内外景占比（按地点/转场标记/情节段落统计）。
- 演员数量：核心演员数（主角/关键反派）与群演估算（群演≈核心×0.6，群众戏或大型场面上调）。

三、宣发（受众与传播层）

- 受众定位：年龄段与圈层（如下沉/一二线、45+中老年/18–35青年、男性/女性），结合题材与人物弧光。
- 上线时间：推荐窗口（工作日晚间/周末黄金/节假日前后），考虑平台热度与排期竞争度。
- 营销方向：主打卖点与素材策略（反转爽点/成长救赎/甜虐情绪/知识信息量/家国叙事），明确切片节奏与标题话术。

四、商业价值（市场潜力评估）

- 题材热度、传播因子、人群覆盖、可生产性（成本可控度）、合规风险。作为综合判断项，可单列为“商业价值指数”。

评分说明（量化维度与权重）

- 总分 100，分项权重：
  - 内容 50（剧情结构 15；角色塑造 15；冲突设置 10；台词质量 10）。
  - 人物饱满度 20（关键词与反差加权，按文本长度归一）。
  - 制作 15（场景数量与内外景占比 8；演员数量结构 7；并结合导演风格匹配作为定性备注）。
  - 宣发 10（受众定位 4；上线时间 3；营销方向 3）。
  - 商业价值 5（题材热度/传播因子/可生产性/合规风险综合）。

具体打分规则（0–100：极弱0–20/偏弱21–40/中等41–60/较强61–80/卓越81–100）：

- 剧情结构（15）：
  - 起承转合完整度（6）：出现“引子/冲突爆发/反转/高潮/结局”等标志词≥5项→≥5分；4项→4分；≤3项→≤3分。
  - 转折密度（4）：每 4k–6k 字至少 1 次关键反转；<1/8k 字降至≤2分；>1/3k 字如逻辑自洽可达4分，否则扣并发散分。
  - 悬念闭合率（3）：铺垫点回收比例≥0.8→3分；0.6–0.79→2分；<0.6→≤1分。
  - 节奏均衡度（2）：场景长度方差适中、段落节拍稳定→2分；过度均一或波动过大→1分。
- 角色塑造（15）：
  - 目标-动机-障碍（5）：主角与关键反派均完整→5；仅主角完整→3–4；缺失→≤2。
  - 弧光与反差（6）：“反转/黑化/洗白/救赎/成长/挣扎/人性”与连接词共现频高、阶段性变化明确→4–6；弱→≤3。
  - 关系网复杂度（3）：关键关系≥4条、存在关系变化≥2次→3；一般→2；简单→≤1。
  - 记忆点（1）：昵称/口头禅/行为记忆点≥2→1；否则0。
- 冲突设置（10）：
  - 类型覆盖（3）：至少覆盖两类（人-人/人-环境/人-自我）→3；单一→≤1。
  - 升级曲线（3）：强度递增≥3阶→3；2阶→2；无升级→≤1。
  - 风险与代价（3）：出现“生命/家庭/事业/名誉”等高风险词并兑现代价→3；虚高或未兑现→1–2。
  - 解决策略多样性（1）：方案≥2种且非“神降解题”→1；否则0。
- 台词质量（10）：
  - 人设匹配（3）：语体一致、口吻贴合→3；偶有违和→2；明显不符→≤1。
  - 信息密度（3）：每 100 字新增有效信息点≥1.5→3；1.0–1.49→2；<1.0→≤1。
  - 金句率与记忆点（2）：每千字≥2句不浮夸→2；仅堆砌或空喊口号→≤1。
  - 生活化/专业度（1）：常识与术语准确→1；错漏→0。
  - 价值观与合规（1）：无不当话术与尺度→1；存在风险→0。
- 人物饱满度（20）：
  - 评分计算：
    - 关键词权重（40%）：反转/黑化/洗白/救赎/成长/挣扎/人性出现频次（每千字标准化）。
    - 连接词权重（30%）：但是/然而/一方面/另一方面/看似/实则/表面/内心共现强度。
    - 反差词对（20%）：正/反面特质同现并与事件绑定（如“温柔/暴烈”、“理性/冲动”）。
    - 事件层级与弧光（10%）：阶段性变化≥3次、与关键节点挂钩。
  - 标签：高（≥67）/中（34–66）/低（＜34）。
- 制作（15）：
  - 场景数量与内外景占比（8）：场景总数、内景≥60%更易控成本；极端外景或大规模群众戏降低可生产性。
  - 演员数量结构（7）：核心角色 4–8 为佳；群演≈核心×0.6 为基线，大场面适度上调。
- 宣发（10）：
  - 受众定位（4）：题材与弧光匹配“下沉/一二线、45+/18–35、男/女”细分标签。
  - 上线时间（3）：工作日晚间/周末黄金/节假日前后依据平台热度与竞品排期。
  - 营销方向（3）：反转爽点/成长救赎/甜虐情绪/知识信息量/家国叙事，明确切片节奏与标题话术。
- 商业价值指数（5）：
  - 题材热度（2）：红果近 1–3 月数据与公开报道；对齐抖音/快手/小红书偏好。
  - 传播因子（1.5）：反转密度、情绪强度、共鸣点、话题词可制作性。
  - 可生产性（1）：场景/演员/内外景比例→成本可控度。
  - 合规风险（0.5）：价值观与尺度安全。

数据口径与量化依据（落地方法）

- 字数与页数：按纯文本统计，docx 转 txt 后去除样式；千字为单位。
- 转折密度：识别“反转/但/然而/却/其实/看似/结果/谁知”等转折触发词，计算每千字频次并人工验证关键反转。
- 悬念闭合率：统计“伏笔/谜团/悬念/未解释线索”与“解释/揭示/回收”，闭合率=回收数/铺垫数。
- 多面性指数：关键词（反转/黑化/洗白/救赎/成长/挣扎/人性）与连接词（但是/然而/一方面/另一方面/看似/实则/表面/内心）加权求和，并按文本长度归一。
- 关系网复杂度：提取称谓/亲属/师徒/上下级/同事等关系边，计算关键节点与关系变化次数。
- 信息密度：有效信息点（推进剧情/塑造人物/建立世界观/埋伏笔）/总字数。
- 风险与代价：识别“生命/家庭/事业/名誉/自由/金钱”等词与兑现事件。
- 场景统计：以地点更换/转场标记/情节段落为场景单位；内/外景按地点词与环境描写判定。
- 演员估算：文本出现的主要角色与关键反派为核心；群演以群众戏标记和场面规模估算（基线=核心×0.6）。
- 宣发口径：受众标签、上线窗口与营销方向对齐平台偏好（红果、抖音、快手、小红书）。

常见问题与改进建议

- 结构不完整：补足“冲突爆发/反转/高潮/结局”等关键段落，控制转折密度在每 4k–6k 字 1 次。
- 人物单薄：增加“连接词驱动的反差事件”，确保目标-动机-障碍齐备，设计弧光节点≥3。
- 冲突弱化：明确风险与代价，避免无成本反转；拉直升级曲线，至少三阶。
- 台词空泛：提高信息密度≥1.5/100字；减少口号与无效寒暄；统一人设语体。
- 制作超预算：提高内景占比与场景复用；压缩群演规模；将大型外景集中处理。
- 宣发不聚焦：二选一主卖点（反转/情绪/知识/家国），统一话术与切片节奏。
- 合规风险：提前清理敏感题材与不当话术，强化价值观正向表达。

评估流程指引

1. 扫描目录、读取 `txt/docx` 文本；`pdf` 无法解析时提示用户提供可解析文本版本。
2. 统计字数与页数，初步判断文量/集数与时代背景、倾向与主题标签。
3. 结构分析：识别转折词与标志段落，计算转折密度与悬念闭合率，评估节奏均衡。
4. 人物分析：抽取角色目标-动机-障碍，计算多面性指数与关系网复杂度，给出饱满度评分与标签。
5. 冲突与台词：评估类型覆盖、升级曲线、风险代价；统计信息密度与金句率，检查语体一致性与合规。
6. 制作估算：统计场景总数与内外景占比、核心/群演数量，给出导演风格建议。
7. 宣发判断：受众定位、上线时间窗口与营销方向，结合平台舆情与投流偏好。
8. 商业价值：综合题材热度、传播因子、可生产性与合规风险，给出指数（0–100）。
9. 输出到 Excel：严格按字段顺序填报；必要时附 3–5 条爆款潜力结论与平台匹配建议。

常见问题解答（FAQ）

- “频向”与“倾向”是否一致？一致，本规则统一按“倾向”。
- 字数统计是否包含标点？包含中文标点但不包含样式与图片。
- 反转密度过高会不会扣分？会；若逻辑不自洽或无代价，视为“无成本反转”，降低结构与冲突评分。
- 商业价值是否影响总分？作为 5 分参考项提供“商业价值指数”，不压过内容与人物评分的主导性。
